# Superstore-Sales-Analysis

Project Overview
The Superstore Sales Analysis project focuses on analyzing retail sales data to understand business performance, customer purchasing behavior, product trends, and regional sales patterns across the United States. The dataset contains nearly 10,000 sales records from 2011 to 2014, covering different product categories, customer segments, shipping modes, and regions.

This project was developed entirely using Microsoft Excel for data cleaning, analysis, visualization, and dashboard reporting. The analysis helps identify top-performing products, profitable regions, sales trends, discount impact on profit, and customer segment performance using Excel formulas, Pivot Tables, Pivot Charts, slicers, and KPI dashboards.

An interactive Excel dashboard was created to visualize important KPIs such as total sales, total profit, top-selling categories, regional performance, customer segment analysis, and monthly sales trends. The project demonstrates practical skills in data analysis, dashboard design, business reporting, and visualization using Excel.
